Design System Philosophy

Zero Deviation Policy

Core Principles

  • Clarity First: Information hierarchy guides user attention
  • Accessibility: WCAG 2.1 AA compliance minimum
  • Mobile-First: 320px+ responsive, touch-friendly
  • Minimalist Elegance: Restraint in color and shadow usage
  • Component Isolation: All components work independently

Implementation Standards

  • Typography: Poppins headers, Quicksand body/buttons/inputs
  • Colors: Orange used sparingly, gray-only icons
  • Shadows: Cards only, no button/form shadows
  • Buttons: Secondary-first philosophy
  • Performance: CSS custom properties, optimized selectors
